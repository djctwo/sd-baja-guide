# San Diego × Baja — Events Guide

Public events guide covering San Diego County and Baja California, Mexico.

**Live site:** https://djctwo.github.io/sd-baja-guide/

## What's in the guide

- ~195 curated events from April 2026 through January 3, 2027
- 7 regions: San Diego · North County · Tijuana · Rosarito · Ensenada · Valle de Guadalupe · Tecate/Puerto Nuevo
- Festivals, concerts, music, food, wine (including 17 individual Vendimia winery cards), art, cultural events, and holiday programming
- Interactive map with filterable event cards
- Countdown to the next upcoming event
- Filters by region, type, and month

## Design

"Barragán by the Sea" — flat color blocks, Talavera tilework, Pacific coast editorial feel. Stucco cream, terracotta, cobalt, and hot-pink palette on Fraunces + DM Sans + Caveat typography.

## Stack

Single-file `index.html` with embedded CSS and JS. Leaflet.js for the map. Deployed via GitHub Pages.

No build step, no database, no framework.
